# 🎨 画像プロンプト置き場

イラスト担当「カエデ」が、各記事のアイキャッチ画像の**生成プロンプト**（Canva AIに貼る指示文）をここに保存します。

## 使い方（社長）
1. カエデが作ったプロンプト（`YYYY-MM-DD-画像プロンプト.md` など）を開く
2. プロンプトを **Canva AI** に貼って画像を生成
3. できた画像を `public/images/hero/<スラッグ>.png` に保存（または編集長に渡す）
4. 編集長が記事の設定に `heroImage:` を追加して組み込み → ブラウザで確認

## トンマナ（統一ルール）
トップページの画像と同じ「手描き水彩・暖色（オレンジ/アンバー）・白背景・文字なし・16:9」で統一する。
