# 📈 SNS分析ログ（X・Threads）

週に1回、社長がX/Threadsのアナリティクスをスクショ → 編集長が下記フォーマットで分析し、ここに追記する。
推移を追い、ツムギ（SNS担当）の投稿方針に反映する。

## 分析フォーマット（毎回これで書く）
- 前週比較（投稿数／閲覧／返信／新規フォロワー）
- 効いた投稿（数字が伸びた投稿とその理由）
- 沈んだ投稿
- 残る課題
- 次の一手（来週ツムギが試すこと）

---

## ✅ 確定した勝ちパターン（ツムギはこれを踏まえて投稿を作る）
- **問いかけ・共感型は効く**：あるある共感の入口→末尾を問いかけ、が返信を生む（返信0→4→12と継続的に増加）
- **伸びるテーマ**：セキュリティ×お役立ち／季節性のあるお金／**AIニュースの3行まとめ**（実ニュース＋ツムギの個人的見立てを添える形に進化させたところ、Xで自己最高の266インプレを記録）
- **フォロワー獲得**：第2回まで0が続いていたが、第3回で新規+8を記録し突破。問いかけ・共感型の継続と返信対応が効いたとみられる
- **Threadsが主戦場**（閲覧数でXの十数倍〜20倍規模のリーチ）。返信・フォロワー増ともThreads側で顕著
